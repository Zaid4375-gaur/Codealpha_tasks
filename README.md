# CodeAlpha Bioinformatics Internship Tasks

This repository contains the tasks completed during my internship at CodeAlpha.

### Tasks Overview:
* **Task 1: DNA/Protein Sequence Analysis** - Performed BLAST analysis to find homologous sequences.
* **Task 2: Multiple Sequence Alignment (MSA)** - Performed MSA using Clustal Omega to highlight conserved regions.
* **Task 3: Protein Structure Prediction** - Used SWISS-MODEL/iCn3D to predict and visualize 3D structures.
